Web-розробка

Завдання курсу – надати базові знання та вміння WEB – розробника, ознайомити з мовами веб-програмування та інструментами, які необхідні для ефективної роботи. Пройшовши цей курс Ви навчитесь створювати WEB – сторінки, володіти сервісами та інструментами та правильно їх використовувати. Також будете вміти додавати інтерактивність сторінкам за допомогою мови Javascript. Цей курс стане міцним фундаментом для впевненого розвитку у професії WEB – розробника.

Тривалість курсу: 3 місяці

Вартість курсу: 12 000

Програма курсу

Основи верстки

  • Знайомство із середовищем розробки
  • Специфіка роботи WEB - програміста
  • Структура HTML - документа
  • Основні теги та коментарі
  • Приклади простої стилізації сторінки
  • Варіанти підключення CSS
  • Основні поняття селекторів
  • Специфічність селекторів
  • Демонстрація верстки WEB -сторінки

Основи програмування

  • Базовий JavaScript
  • Демонстрація інтерактивності на JavaScript
  • Базові можливості jQuery
  • Демонстарція популярного фреймворку Angular.js
