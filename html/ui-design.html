UI дизайн

UI дизайнер займається проектуванням інтерфейсів на основі даних, зібраних від замовників, користувачів, цілей бізнесу і свого експертного досвіду. Головною метою UI дизайну є розробка візуальної складової продукту (інтерфейс повинен бути констистентним, зрозумілим та легким в візуальному сприйнятті).

Тривалість курсу: 3 місяці

Вартість курсу: 12 000

Програма курсу

  • Основи юзабіліті
  • Аналіз конкурентів
  • Інтерв’ю з стейкхолдерами та юзерами
  • Методологія JTBD. Як писати Job Stories. Принцип KISS. Гештальт принципи в дизайні
  • Типографіка. Найпопулярніші шрифти. Способи підбору шрифтів
  • Колористика. Як збирати кольорові палітри
  • Форми в інтерфейсі. Логіка створення форм. Data Design, Data Visualization
  • Анімації в інтерфейсі
  • Розробка презентації для Behance в форматі Case Study
