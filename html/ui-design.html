UI дизайн

UI дизайнер займається проектуванням інтерфейсів на основі даних, зібраних від замовників, користувачів, цілей бізнесу і свого експертного досвіду. Головною метою UI дизайну є розробка візуальної складової продукту (інтерфейс повинен бути констистентним, зрозумілим та легким в візуальному сприйнятті).

Тривалість курсу: 3 місяці

Вартість курсу: 12 000

Програма курсу

  1. Основи юзабіліті
  2. Аналіз конкурентів
  3. Інтерв’ю з стейкхолдерами та юзерами
  4. Методологія JTBD. Як писати Job Stories. Принцип KISS. Гештальт принципи в дизайні
  5. Типографіка. Найпопулярніші шрифти. Способи підбору шрифтів
  6. Колористика. Як збирати кольорові палітри
  7. Форми в інтерфейсі. Логіка створення форм. Data Design, Data Visualization
  8. Анімації в інтерфейсі
  9. Розробка презентації для Behance в форматі Case Study
