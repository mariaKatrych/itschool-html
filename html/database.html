Бази даних

Мета курсу — надати слухачам знання та навички, необхідні для освоєння технологій баз даних або розширення своїх знань в цій області. Курс описує основні концепції баз даних, включаючи такі поняття як типи баз, мови роботи з базами даних, дизайн баз даних. Цей курс призначений для фахівців, які тільки починають працювати з базами даних.

Тривалість курсу: 3 місяці

Вартість курсу: 12 000

Програма курсу

  1. Реляційна база даних. Дизайн бази даних. Діаграми
  2. Побудова звязків між сутностями
  3. Робота з обєктами баз даних : створення таблиць, представлень, робота з констрейнтами
  4. Індекси
  5. Написання збережених процедур, функцій, тригерів. Динамічний SQL
  6. Тимчасові таблиці. Курсори
  7. Закріплення знань T-SQL. Знання основних операторів, вміння застосувати навички при роботі над проектом. Агрегаційні, віконні функції
  8. Вивчення основних компонентів SQL Server Management Studio
  9. Побудова сховища даних
  10. SSIS. Побудова проекту для заливки даних з файлів у базу даних. Забезпечення інрементальної заливки даних
  11. Аналітика. Побудова звітів за допомогою SSRS, TABLEAU
  12. Закріплення теоретичних знань
